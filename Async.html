Asynchronous Javascript

      If we use synchronous way to handle logics that take time it blocks the for the
      other codes below that specific time taking code so we use setTimeOut function 
      which accepts two parameters the function to be executed and the time it is going 
      to be delayed that is Asynchronous

How do we use/consume Asynchronous variables in our synchrounous way

      if variable gets its value in the asynchronous setTimeout function we use it in synchronous below 

      
1.callback functions


      function orderSome(eatOrder) {
        console.log("order your something");
        setTimeout(() => {
          const order = "#";
          console.log(`your ${order} is ready`);
          eatOrder(order);
        }, 2000);
      }

      function eatOrder(order) {
        console.log(`eat your ${order}`);
      }

      orderSome(eatOrder);
      console.log("After callback");

      we access the variables inside the setTimeout using callback function this means the eatOrder() is the callack function.
