byClassName and byTagName method uses addressing method this means the values get updated live and checks the previous code that are affected by the change we made so the final result looks like the function that we want to make happens jumping one element

# Solution

Solution 1 — Loop backwards When looping backward, index shifting doesn't break things. 

          
              let items =document.getElementsByClassName("yellow"); 
              for (let i = items.length -1; i >= 0; i--) { 
              items[i].classList.remove("yellow"); 
            }
Solution 2 — Convert to Array (best practice) Make a static copy first. 
            let items =[...document.getElementsByClassName("yellow")];
            items.forEach((el) =>{ el.classList.remove("yellow"); });
            Now DOM changes won't affect the loop.
